Программа поддерживает взаимодействие с сервером несколький клиентов в рамках сети.
Рекомендуется запускать сервер на Windows, а клиентов на Linux, для наиболее удобного тестирования

Для сборки программы на Linux предусмотрен makefile. 
> make -f makefile

Для сборки программы на Windows рекомендуется использовать Visual Studio
(Все необходимые компоненты размещены в соответствующей папке)

Для запуска программы необходимо указать соответствующий ключ:

--server(сокр -s) запуск сервера

--client(сокр -c) ip запуск клиента, где ip - ip адрес сервера(команды ifconfig(LIN), ipconfig(WIN))

База данных хранится в соответствуюзей папке database
